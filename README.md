# Traffic Congestion Prediction

A machine learning project developed as part of the Uber × upGrad MentorMind Program to analyze traffic patterns and predict traffic congestion using historical transportation data.

## Problem Statement

Urban traffic congestion affects travel time, fuel consumption, and city infrastructure. The objective of this project was to analyze historical traffic data, identify patterns, and build a predictive model capable of forecasting traffic conditions.

## Dataset

The project uses historical traffic datasets containing information such as:

* Traffic volume
* Junction information
* Date and time
* Peak and non-peak hours
* Historical congestion patterns

## Project Workflow

### 1. Data Cleaning

* Handled missing values
* Removed inconsistencies
* Prepared data for analysis

### 2. Exploratory Data Analysis (EDA)

* Analyzed traffic trends across different junctions
* Identified peak-hour congestion patterns
* Visualized traffic distribution and seasonal behavior
* Compared traffic movement across locations

### 3. Feature Engineering

* Extracted time-based features
* Created meaningful variables for model training
* Prepared datasets for machine learning workflows

### 4. Model Development

* Built and evaluated machine learning models for traffic prediction
* Compared model performance using standard evaluation metrics
* Optimized model accuracy through iterative improvements
